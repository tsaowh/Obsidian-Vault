		- 債權人占有
			- 因 修車 而 占有 車
		- **債權之發生 與 該動產 有牽連關係**
			- 債權之發生
				- ==**490 承攬契約 所生之 修車費用**==
			- 該動產
				- 車

		- 若 因侵權行為或其他不法之原因而占有
		- 則 不適用 留置權

		- 則 亦不適用 留置權
