### 財產法與身份法之適用性

- 民法一些制度跟條文==**可用在財產法（身份法）但在身份法（財產法）時會行不通**==，因為目的不同
	- 民法行為（法律行為）之==**能力（指資格及地位）**== 原則為==**財產法**==制度
		- 18歲成年具**完全行為能力** 為交易行為不需父母同意
			- **保護未成年**
			- **保護交易安全**
		- ==15（**監護宣告**制度）== 受監護宣告之人，==**無行為能力**==。
			- ==**保護交易安全**==
				- 其**法律行為無效 防範其財產被騙走**
			- 其法律行為由監護人為之（如同法定代理人）
		- ==***身份行為原則無能力（指資格及地位）限制***==
			- **結婚（身份契約）**
				- **不得代理**，只要意識清晰有**意思能力**即可
			- **遺囑能力**
				- 民法1186條**16歲未成年**即可為之
	- 民法行為（法律行為）之==**附款（有條件跟期限）**== 原則為==**財產法**==制度
		- ==***身份行為原則無附款（有條件跟期限）限制***==，因為身份行為特重==***安定性*** ==
			- eg: 一般不會再結婚時就約定離婚的條件及期限
	- ==**代理制度**==原則為==**財產法**==制度
		- 統一企業聘請員工為代理人，為交易行為其效益歸統一企業
			- 擴大本人的經濟生活法律關係 
			- 為財產法制度
		- ==***身份行為原則不可代理***==，因為身份行為特重==***真意***== 
			- eg: 結婚

### 民法3大原則

- 民法3大原則（會隨社會變遷些微調整，民法為私法具**私法自治**：人民自己管自己，立法者不會介入人民之間的法律關係）
	- ==所有權**絕對**原則（物權原則 民法765）==：沒有很重要 但 **絕對權** 為所有交易行為的基礎
		- 只要有該物所有權，可==**自由使用收益處分**== 排除他人干涉
		- **==修正所有權絕對==**（隨社會變遷些微調整）
			- 要符合誠信原則、不能干涉他人權益、兼顧公共利益
			- 所有權在部分情況下應==社會化（基於公益）==
				- 民法物權篇之==**相鄰關係**==
					- 土地相鄰，不動產之間的利用關係，**==課予容忍義務==**
				- ==**土地徵收**==
					- 行政法：基於 重大公共利益，直接透過 **行政處分** 剝奪人民 所有權（侵害私權）
				- **==公用地役==關係**（於物權說明） 
					- eg: ==**既成道路**==
					- 所有權不會被剝奪
					- 但所有權需在 特定的目的範圍內 供 不特定多數人 使用（不能將土地圍起來）
	- ==契約自由原則（債權原則）==
		- **主觀等價**（非客觀）：每個標的物價值是主觀認定
		- **==修正主觀等價==**（隨社會變遷些微調整）
			- 主觀等價不公平之時
				- 締約能力、條件、地位不一樣 eg: 老闆開出的薪資條件過低（迫害勞工的權益）
			- ==**特別民法（實現契約正義）**==
				- **勞基法**（規定最低工資、基本工時）、**消保法**
				- 勞動、消費者契約 ==**違反上述 強行法規：無效**==
					- 強行法規 vs 任意法規
						- ==**強行**法規==
							- ==**不能**== 由當事人用 ==**約定排除或變更**== 的法律規定
							- 分為
								- ==**禁止**規定==
									- 不能 買賣毒品
								- ==**強制**規定==
									- 勞基法 最低工資
									- 一定要 登記 才生效
							- eg
								- 不能 買賣毒品
								- 勞基法 最低工資
								- 一定要 登記 才生效
						- ==**任意**法規==
							- ==**能**== 由當事人用 **約定排除或變更** 的法律規定
							- eg
								- 利息多少（在合法範圍內）
	- ==**過失責任主義（侵權行為原則（184--1）、債務不履行原則）：最重要**==
		- 損害賠償**以過失為前提**
			- 非造成損害就一定要賠，==**有過失可歸責**==的情形下才需要負責
				- eg: 合於 速限 及 所有交通法規駕駛則 不一定要負責
		- ==**修正原告（被害人、債權人）負舉證責任**==
			- 184--1 前段 侵權行為過失之要件，==依 **民訴 277（舉證責任分配之原則）** 原則 由原告（被害人、債權人）負舉證責任==
				- 184（獨立侵權行為之責任）
					- 184--1 因故意或過失，不法侵害他人之權利者，負損害賠償責任。故意以背於善良風俗之方法，加損害於他人者亦同。
						- 故意
							- 肯定 賠
						- 過失
							- 依 **民訴 277（舉證責任分配之原則）** 原則 由原告（被害人、債權人）負舉證責任
				- 民訴 277（舉證責任分配之原則）當事人主張有利於己之事實者，就其事實有舉證之責任。==**但法律別有規定，或依其情形顯失公平者，不在此限**==。
					- ==積極事實 原則由 主張之人 負舉證責任==
				- 某種程度上對被告（加害人、債務人）公平
				- 但有些侵權情況，==被害人無具備專業知識（**證據偏在性**），由被害人舉證並不公平==
					- 汽車零件損壞（機械專業）
					- 手術
			- ==**推定**==過失責任（==**184--2**==）
				- 184--2 違反保護他人之法律，致生損害於他人者，負賠償責任。==（**推定過失**）但能證明其行為無過失者，不在此限==。
					- 但...不在此限：為推定之例外，依訴訟法之規範說，主張例外之人要負舉證責任
				- 只要符合推定過失之規定，==直接當作**加害人有過失**，加害人需**提出自己無過失**之證據==
				- 由被告（加害人、債務人）舉證自己無過失
				- ==對被害人有利==
			- ==**無**==過失責任（==**很少 eg: 消保法 7、國賠法 3**==）
				- 加害人==就算沒有過失亦要負責任==
					- 只要證明損害為對方造成，不用證明是否有過失，可直接請求損害賠償
				- ==最嚴格==之責任
				- ==對被害人最好==
		- ==**過失責任** 程度（**要不要討論 過失 程度**）vs **過失** 程度（**有沒有達到 注意義務**）==
			- 過失責任 程度（要不要討論 過失 程度）
				- ==**過失**責任（**預設**）==
					- 須討論 過失 程度
					- 由 ==**被害人 舉證**==
				- ==**推定過失**責任（對被害人有利）==
					- 須討論 過失 程度
					- **but 由 ==加害人 舉證==**
				- ==**無過失**責任（對被害人最好）==
					- ==**不須討論**== 過失 程度
			- 過失 程度（有沒有達到 注意義務）
				- ==**抽象輕**過失（**預設**，嚴格）==
					- 注意義務程度 ==**善意管理人**==
				- ==**具體輕**過失==
					- 注意義務程度 ==**自己**==
				- ==**重大**過失（寬鬆）==
					- 注意義務程度 ==**一般人**==
			- **過失責任 + 抽象輕過失**
				- **一般侵權（184--1）**
					- 184（獨立侵權行為之責任）
				- 借用人**保管**借用物
					- 468（借用人之保管義務）
				- 質權人保管質物
					- 888（質權人之注意義務）
			- **過失責任 + 具體輕過失**
				- 與**自己事務**同一注意型**保管**
					- **223（具體輕過失之最低責任）**
				- 質權人**收取孳息**
					- 890（孳息收取人之注意義務及其抵充）
			- **過失責任 + 重大過失**
				- **223（具體輕過失之最低責任）** 應與處理自己事務為同一注意者，**如有重大過失，仍應負責**
					- **避免 他人 拿 一般人** 注意義務程度 **當 自己** 注意義務程度
			- **推定過失責任 + 抽象輕過失**
				- **違反保護他人法律（184--2）**
					- 184（獨立侵權行為之責任）
				- **僱用人、動物、工作物、商品、動力車輛等責任**
					- 188（僱用人責任）
						- 受僱人因執行職務，不法侵害他人權利時，由**僱用人與行為人==連帶==** 負損害賠償責任；僱用人能證明選任、監督已盡相當注意者，得免責。
					- 190（動物責任）
						- 動物加損害於他人者，由其占有人負損害賠償責任；但已依動物種類、性質為相當注意之管束，或即使相當注意仍難避免者，不在此限。
					- 191（工作物責任）
						- 土地上建築物或其他工作物致他人權利受損時，由工作物所有人負賠償責任；但其能證明設置、保管無欠缺，或損害非因欠缺所致，或已盡相當注意者，得免責。
					- 191-1（商品責任）
						- 商品製造人因商品通常使用或消費所致他人損害，原則上負賠償責任；但能證明生產、製造、加工、設計無欠缺，或損害非因該欠缺所致，或已盡相當注意者，不在此限。
					- 191-2（動力車輛責任）
						- 汽車、機車或其他非依軌道行駛之動力車輛，在使用中加損害於他人者，駕駛人原則上應負賠償責任；但已盡相當注意者，不在此限。
			- **無過失責任**
				- **消保法 7（企業經營者就其商品或服務所應負之責任）**
					- 企業提供的商品或服務要有合理安全性
					- 違反致損時要賠，而且證明無過失也不一定能全免
				- **國賠法 3（公共設施責任）** 公共設施因設置或管理有欠缺，害人民生命、身體、人身自由或財產受損害時，國家要賠。

### 民法過失類型

- **刑法**總則**過失**
	- **無意識**過失：按其情節，應注意能注意，而未注意
		- 應預見，but 沒有預見
		- 即 疏忽
	- **有意識**過失
		- 有預見，but 相信不會發生
	- **未必故意**
		- 有預見，but 就算發生也無所謂

- ==**民法**過失抄**刑法之無意識**過失==
	- ==**不同之處：具有標準線（善意管理人（抽象輕）、自己（具體輕）、一般人（重大））**==
		- 高於標準：無過失
		- 低於標準：有過失
	- 民法過失類型
		- ==**抽象輕**==過失（大家標準皆同 不會因人而異）
			- **注意義務程度**：==**善意管理人**==（為法律抽象概念：形容有相當知識、經驗、勤勉、負責的人 eg: 95分 要求==**嚴格**==）
			- ==**民法所有條文只要沒有特別提皆以此標準**== 都為一樣標準才公平 eg: 交通法規
		- ==**具體輕**==過失
			- **注意義務程度**：==**自己**==（平常之義務標準 eg: 80分 要求==**適中**==）
			- ==**535 委任契約**==
				- **受任人**處理委任事務，應**依委任人**之指示，並與處理**自己**事務為同一之注意，其**受有報酬**者，應以**善良管理人**之注意為之。
					- ==無對價（無償）==：與處理==自己==事務為同一之注意
					- ==有對價（有償）==：受有報酬者，應以==善良管理人==之注意為之 eg: 律師幫客戶處理訴訟，收取律師費用
				- 過失責任主義之立法（220（債務人責任之酌定） 不重要）
					- 220--1 債務人就其**故意或過失**之行為，應負責任。
						- ==故意==之責任
						- **故意責任比重大過失要求==更寬鬆==**
						- 故意本就不用負過失責任（因為已故意），**只需就故意行為負責**
							- 故意若賠：理所當然
							- 過失若賠：需討論注意義務程度
						- **民法原則不採故意責任**
					- 220--2 ==過失==之責任，依事件之特性而有輕重，如其事件==**非予債務人以利益者**，應**從輕**酌定。==
						- 非予債務人以利益者
							- 無對價（無償）
						- ==**535 委任契約 為 220--2 之具體化規定**==
		- ==**重大**==過失（**常與==故意==並列**，當作例外不保護或排除）
			- **注意義務程度**：==**一般人**==（eg: 60分 要求==**寬鬆**==）
			- ==**410 贈與契約**==
				- **贈與人**僅就其**故意或重大過失**，對於受贈人負**給付不能**之責任。
					- 贈與契約 中，**贈與人 除 贈與 外，沒給 贈與人 任何利益**，所以其注意義務程度**降到重大過失**
			- ==**434 失火責任**==
				- 租賃物因承租人之重大過失，致失火而毀損、滅失者，承租人對於出租人負損害賠償責任。
					- 因**重大公共利益（保護弱勢（承租人））** 目的，所以其注意義務程度**降到重大過失**
			- ==立法者須**清楚載明**注意義務程度**才會降到重大過失**==
				- **民法原則不採故意責任**
