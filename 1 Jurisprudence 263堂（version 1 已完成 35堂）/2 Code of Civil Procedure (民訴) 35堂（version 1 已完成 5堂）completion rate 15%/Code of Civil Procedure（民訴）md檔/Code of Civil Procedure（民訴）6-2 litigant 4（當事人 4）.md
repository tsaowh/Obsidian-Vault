		- 法律依據
			- 44-3（提起不作為訴訟之權）
				- 44-3-1 以公益為目的之社團法人或財團法人，經其目的事業主管機關許可，於章程所定目的範圍內，得對侵害多數人利益之行為人，提起不作為之訴。
				- 44-3-2 前項許可及監督辦法，由司法院會同行政院定之。
				- 立法目的
					- 因 公害、商品瑕疵等事故，有時具 繼續性、隱微性、擴散性
					- 故 受害人 常不知、無力獨自訴請排除侵害，使 社會大眾權益（公益）持續受損無從制止
					- 故 須擴大 公益法人 功能
						- 公益法人
						- 經其目的事業主管機關許可，於章程所定目的範圍內
						- 得對侵害多數人利益之行為人，提起不作為之訴

			- 固有權說
				- 程序法上
					- 44-3（提起不作為訴訟之權）
				- 實體法上
					- 公益法人 借用 程序法上 44-3（提起不作為訴訟之權），作為 實體法上 之 請求權基礎
						- 被害人 本身即有 體法上 之 請求權基礎
							- 184（獨立侵權行為之責任）
							- 767（所有權之保護－物上請求權）
							- 793（氣響侵入之禁止）
				- 訴訟標的 即 該公益法人 固有獨立 之 不作為請求權
					- 訴訟標的 = 訴之聲明（原告請求的內容） + 原因事實（主張請求的具體理由、法條依據）
					- 因 訴訟標的 即 該公益法人 固有獨立 之 不作為請求權，故 非 訴訟擔當
					- 若 公益法人 依 44-3（提起不作為訴訟之權）起訴
						- 若 其他公益法人、被害人 另行 起訴

							- 無 重複起訴禁止（情境：訴訟繫屬 中（判決未確定）時）
								- 即 無違反 253（一事不再理）
								- 依 舊同一事件說（訴之三要素）判斷是否為 同一事件
									- 當事人 是否 同一
										- 因 非 訴訟擔當
										- 故 公益法人 與 其他公益法人、被害人 不同
									- 訴訟標的 是否 同一
										- 因 非 訴訟擔當
										- 故 公益法人 之 不作為請求權 與 其他公益法人、被害人 之 不作為請求權 不同
									- 訴之聲明 是否 相同、相反、可待用
									- 雖 訴之聲明 可能相同，當事人、訴訟標的 不同，即 非 同一事件
										- 一要素 不同，即 非 同一事件
								- 因 仍 訴訟繫屬 中，故 尚毋需討論 既判力
							- 無 受前判決 判決效力擴張所及（情境：訴訟繫屬 已消滅（判決確定）時）
								- 因 非 訴訟擔當、訴訟標的 不同
								- 故 既判力 僅及於 公益法人 與 被告
									- 未及於 其他公益法人、被害人
						- but 仍應儘量 併案審理，依 67-1（訴訟事件及進行程度通知利害關係人之第三人）通知 法律上利害關係人，將其引入訴訟
