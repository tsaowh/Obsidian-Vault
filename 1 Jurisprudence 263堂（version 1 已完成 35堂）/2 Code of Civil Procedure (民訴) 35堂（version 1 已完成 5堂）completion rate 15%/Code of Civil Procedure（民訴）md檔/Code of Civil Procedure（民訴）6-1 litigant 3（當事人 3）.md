- 當事人適 格判斷標準
	- 原則（不常考，例外（訴訟擔當）較常考）
		- 以 ==**實體權利義務 之 歸屬主體**== 判斷 該當事人適格與否
		- 不同訴訟類型 之 當事人適格 判斷標準
			- ==**給付之訴（要 他方當事人 做與不做 某事（雙方））**==
				- 以 **訴訟 方式行使 請求權**
				- ==**給付之訴 當事人適格 判斷標準（＊此類型較常考）**==
					- ==**以 原告起訴主張 為斷**==
			- ==**確認之訴（要 法院 判定 存在或不存在（有或沒有） 某法律關係）**==
				- ==**確認之訴 當事人適格 判斷標準**==
					- ==**以 有無確認利益 為斷**==
			- ==**形成之訴（要 法院 變更 某法律關係（單方））**==
				- 以 **訴訟 方式行使 形成權**
				- ==**形成之訴 當事人適格 判斷標準**==
					- ==**以 法律或法理 為斷**==
	- ==**例外（＊＊＊較常考，難）**==
		- ==**訴訟擔當**==
			- 形式當事人 與 實質當事人 為 不同人
				- 一般 情形
					- 形式當事人 與 實質當事人 為 同一人
				- 訴訟擔當 情形
					- 形式當事人 與 實質當事人 為 不同人
			- 第三人（擔當人、形式當事人）就 實體上 非屬己之權利義務 取得 當事人適格
				- 以自己名義 為 真正權利義務歸屬主體（被擔當人、實質當事人）進行訴訟
				- 判決效力 及於 擔當人、**被擔當人**
					- 401（既判力之主觀範圍）
						- 401--1 確定判決，除當事人外，對於訴訟繫屬後為當事人之繼受人者，及為當事人或其繼受人占有請求之標的物者，亦有效力。
							- 當事人
								- 形式當事人
									- 對 法院 而言，程序上亦為 通知 形式當事人 開庭
						- 401--2 對於為他人而為原告或被告者之確定判決，對於該他人亦有效力。
							- 訴訟擔當
							- 該他人
								- 被擔當人
						- 401--3 前二項之規定，於假執行之宣告準用之。
				- 由 擔當人 進行訴訟，判決效力 及於 擔當人、**被擔當人**
			- 擔當人 可為 原告 或 被告，以 原告 為例
				- 甲 告 丙
					- 主張 乙的權利
						- 訴訟標的 = 訴之聲明（原告請求的內容） + 原因事實（主張請求的具體理由、法條依據）
					- 甲
						- 擔當人、形式當事人
					- 乙
						- 被擔當人、實質當事人
			- 訴訟擔當 兩大效力
				- 重複起訴禁止
					- 253（一事不再理）當事人不得就已起訴之事件，於訴訟繫屬中，更行起訴。
					- 重複起訴 判斷標準
						- 實務見解
							- 舊同一事件說
								- 訴之三要素
									- 當事人
										- 原告 或 被告
										- 是否 相同
											- 形式當事人 相同
											- 實質當事人 相同
									- 訴訟標的
										- 訴訟標的 = 訴之聲明（原告請求的內容） + 原因事實（主張請求的具體理由、法條依據）
										- 在此基本上應該是指 法條依據
										- 是否 相同
									- 訴之聲明
										- 是否 相同、相反、可代用
								- 檢視 訴之三要素 後，才能決定是否為 同一事件
					- 若 違反 253（一事不再理），則 後訴法院（裁判 重複起訴 之法院）依 249--1-7 裁定駁回
						- 249--1-7 當事人就已繫屬於不同審判權法院之事件更行起訴、起訴違背第二百五十三條、第二百六十三條第二項之規定，或其訴訟標的為確定判決效力所及。
							- 253（一事不再理）
							- 263（訴之撤回效力）
								- 263--1 訴經撤回者，視同未起訴。但反訴不因本訴撤回而失效力。
									- 反訴
										- 被告 反過來 起訴 原告
								- 263--2 於本案經終局判決後將訴撤回者，不得復提起同一之訴。
							- 不同審判權法院
								- 同一審判權法院
									- 以 管轄競合、253（一事不再理）處理 
										- 基本上於 管轄競合規則 已處理完畢
										- 依 28（移送訴訟之原因及程序）裁定移送
											- 28--1 訴訟之全部或一部，法院認為無管轄權者，依原告聲請或依職權以裁定移送於其管轄法院。
								- 不同審判權法院
									- 僅能以 253（一事不再理）處理
										- 依 249--1-7 裁定駁回
							- 或其訴訟標的為確定判決效力所及
								- 已被確定判決處理過（既判力）
				- 判決效力（尤指 既判力）擴張（主觀範圍擴張）
					- 主觀範圍擴張
						- 程序法上 主觀 為 主體
