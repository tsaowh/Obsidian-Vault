	- 法規
		- 41（選定當事人之要件及效力）
			- 41--1 多數有共同利益之人，不合於前條第三項所定者，得由其中選定一人或數人，為選定人及被選定人全體起訴或被訴。
				- 前條第三項
					- 40--3 非法人之團體，設有代表人或管理人者，有當事人能力。
				- 不合於前條第三項所定者
					- 即 多數有共同利益之人 非 40--3 有當事人能力 之 非法人團體
				- 起訴
					- 為 原告
				- 被訴
					- 為 被告
			- 41--2 訴訟繫屬後，經選定前項之訴訟當事人者，其他當事人脫離訴訟。
			- 41--3 前二項被選定之人得更換或增減之。但非通知他造，不生效力。

			- 訴訟經濟（簡化訴訟）

			-  選定人 須為 多數有共同利益之人
				- 該數人 符合 53（共同訴訟之要件）之 53---1、53---2
					- 53（共同訴訟之要件）二人以上於下列各款情形，得為共同訴訟人，一同起訴或一同被訴：
						- 53---1 為訴訟標的之權利或義務，為其所共同者。
							- 即 對該數人而言 訴訟標的之權利或義務 相同
						- 53---2 為訴訟標的之權利或義務，本於同一之事實上及法律上原因者。
							- 即 對該數人而言 訴訟標的之權利或義務 雖不相同，but 本於同一之事實上及法律上原因
				- 本於同一之事實上及法律上原因
					- 訴訟資料重要部分 共通
					- 主要攻擊防禦方法 共通
				- eg
					- 同一公害事件 等 accident 之 多數被害人
			- 多數有共同利益之人 非 40--3 有當事人能力 之 非法人團體
				- 若 為 40--3 有當事人能力 之 非法人團體
				- 則 以 非法人團體 之 代表人或管理人 起訴 或 被訴
				- 若 該非法人團體 未符合 當事人能力 之 要件
					- 設有 代表人或管理人 者
					- 名稱
					- 事務所或營業所

					- 獨立財產

	- 選定行為
		- 選定行為 即 授予 訴訟實施權 予 被選定人
		- 選定行為 屬 訴訟行為，故 選定人 應具備 訴訟能力
			- 45（訴訟能力）能獨立以法律行為負義務者，有訴訟能力。
		- 為 擴大 選定當事人制度 功能，允許 下列選定方式，為 選定人 及 被選定人 全體起訴 或 被訴
			- 分組選定
				- 共同利益人 分組選定 不同 擔當人
				- eg
					- A1~A100 為 共同利益之多數人
						- A1~A90 選定 A1 為 被選定人（擔當人）
						- A91~A100 選定 A91 為 被選定人（擔當人）
					- 即
						- A1 以自己名義 為 當事人，為 A1~A90 進行訴訟
						- A91 以自己名義 為 當事人，為 A91~A100 進行訴訟
						- 共 2 位 當事人
			- 一部選定
				- 僅由 部分共同利益人 選定 一或數擔當人
				- 與 未參與選定之其他共同利益人
				- eg 
					- A1~A100 為 共同利益之多數人
						- A1~A90 選定 A1 為 被選定人（擔當人）
						- A91~A100 不願選定，選擇 以自己名義 為當事人
					- 即
						- A1 以自己名義 為 當事人，為 A1~A90 進行訴訟
						- A91~A100 以自己名義 為 當事人，為 A91~A100 進行訴訟
						- 共 11 位 當事人

				- 41（選定當事人之要件及效力）
					- 41--2 訴訟繫屬後，經選定前項之訴訟當事人者，其他當事人脫離訴訟。
				- 即使 之後 選定人 去世 或 喪失能力，亦不會影響 被選定人資格
				- 依 401--2 確定判決效力 會及於 實質當事人
					- 401（既判力之主觀範圍）
						- 401--2 對於為他人而為原告或被告者之確定判決，對於該他人亦有效力。
			- 依 44--1 選定人 得限制 被選定人 為 捨棄、認諾、撤回、和解，依 44--2 選定人中一人 所為限制，其效力 不及於 他選定人 
				- 捨棄
					- 提出之當事人 為 原告，法院應判決原告敗訴
				- 認諾
					- 提出之當事人 為 被告，法院應判決被告敗訴
				- 捨棄、認諾、撤回、和解 皆對 選定人 不利，故 選定人 得依 44（選定當事人為訴訟行為之限制）限制 被選定人
				- 44（選定當事人為訴訟行為之限制）
					- 44--1 被選定人有為選定人為一切訴訟行為之權。但選定人得限制其為捨棄、認諾、撤回或和解。
					- 44--2 選定人中之一人所為限制，其效力不及於他選定人。
		- 被選定人資格
			- 被選定人（擔當人）是否有 訴訟實施權
				- 屬 當事人適格 議題
			- 被選定人 喪失資格 處理方式
				- 43（選定當事人喪失其資格之救濟）第四十一條之被選定人中，有因死亡或其他事由喪失其資格者，他被選定人得為全體為訴訟行為。
				- 172（當然停止－喪失一定資格或死亡）
					- 172--1 本於一定資格以自己名義為他人任訴訟當事人之人，喪失其資格或死亡者，訴訟程序在有同一資格之人承受其訴訟以前當然停止。
					- 172--2 依法被選定為訴訟當事人之人全體喪失其資格者，訴訟程序在該有共同利益人全體或新被選定為訴訟當事人之人承受其訴訟以前當然停止。
		- 當事人訴之聲明 與 法院判決主文 表現方式
			- 實務見解
				- 多數選定人 各有獨立 實體法 請求權（請求 損害賠償等），故 訴之聲明（原告請求的內容）應分別記載，法院判決主文 亦應分別記載
					- eg 
						- 訴之聲明
							- 給付 某甲 若干元、某乙 若干元、某丙 若干元...
			- 有學者認為
				- 若當事人 難以 個別證明 損害賠償金額，則 選定當事人 請求 損害賠償 之訴訟
					- 類推適用 44-1-2 總額裁判
						- 44-1（選定法人之要件）
							- 44-1-1 多數有共同利益之人為同一公益社團法人之社員者，於章程所定目的範圍內，得選定該法人為選定人起訴。
							- 44-1-2 法人依前項規定為社員提起金錢賠償損害之訴時，如選定人全體以書狀表明願由法院判定被告應給付選定人全體之總額，並就給付總額之分配方法達成協議者，法院得不分別認定被告應給付各選定人之數額，而僅就被告應給付選定人全體之總額為裁判。
								- 並就給付總額之分配方法達成協議者
									- 分配協議（前提）
					- 立法目的
						- 訴訟經濟
				- 聲明（訴之聲明）之拘束性
					- 若 當事人 聲明 請求損害總額
					- 則 即便 法院 調查證據 後 認為可 個別認定當事人損害金額，法院 僅得闡明之
					- 若 經法院闡明後，當事人 仍不更正 聲明
					- 則 法院 仍不得 為 個別當事人損害金額之裁判
				- 因 上述 訴訟經濟方式、聲明之拘束性 關係到 選定人 實體法上權利（請求範圍）
				- 故 應 目的性限縮解釋 44--1 本文
					- 44（選定當事人為訴訟行為之限制）
						- 44--1 被選定人有為選定人為一切訴訟行為之權。但選定人得限制其為捨棄、認諾、撤回或和解。
					- 即 44--1 之 為一切訴訟行為之權 不包括 於聲明表明損害總額
						- 被選定人 若欲為之，應得 選定人 特別授權
