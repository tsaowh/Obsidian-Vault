							- 物權為絕對權，只要對方行為影響 其權利行使，即可排除侵害
							- 不損害發生為要件 之請求權

				- 甲 可以下列 請求權基礎 對 乙 請求停止排放 工業廢水
					- 767--1 中段
						- 請求除去妨害
