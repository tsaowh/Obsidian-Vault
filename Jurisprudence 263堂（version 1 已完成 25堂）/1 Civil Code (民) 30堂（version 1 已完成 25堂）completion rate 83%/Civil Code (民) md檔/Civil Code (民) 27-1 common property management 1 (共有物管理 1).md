			- 820--3 前二項所定之管理，因情事變更難以繼續時，法院得因任何共有人之聲請，以裁定變更之。
			- 820--4 共有人依第一項規定為管理之決定，有故意或重大過失，致共有人受損害者，對不同意之共有人連帶負賠償責任。
			- 820--5 共有物之簡易修繕及其他保存行為，得由各共有人單獨為之。
