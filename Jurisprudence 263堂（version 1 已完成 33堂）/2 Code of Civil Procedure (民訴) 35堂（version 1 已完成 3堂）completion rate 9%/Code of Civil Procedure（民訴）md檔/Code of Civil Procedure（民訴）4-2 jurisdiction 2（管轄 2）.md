				- 10--1 因**不動產之物權**或其**分割**或**經界**涉訟者，**專屬不動產所在地**之法院管轄。
					- 專屬管轄

						- 確認之訴

						- 不動產物權 為標的 之訴
						- eg
							- 不動產物權
								- 767 三種請求權 皆屬 物上請求權，亦 屬 10--1 之 專屬管轄
