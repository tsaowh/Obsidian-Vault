				- 787--1 **土地因與公路無適宜之聯絡，致不能為通常使用**時，==**除**==因土地所有人之任意行為所生者外，土地所有人**得通行周圍地以至公路**。

					- **償金**
						- 用於**相鄰**關係
						- **不當得利 與 侵權行為** 所取得之**賠償總額**
