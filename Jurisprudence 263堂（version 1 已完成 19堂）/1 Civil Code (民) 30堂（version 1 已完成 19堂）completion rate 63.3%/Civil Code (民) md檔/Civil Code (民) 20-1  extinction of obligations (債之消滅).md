
- ==**債之移轉（有三類型）**==
	- ==**共通點**==
		- 僅為 **==主體==變更**，原先債之 **==同一性（內容）== 不改變**
	- ==**類型**==
		- eg 
			- 甲 車 345 買賣契約 乙
			- 甲 對 乙 有 **348 出賣人之移轉財產權及交付標的物之義務（甲之義務，乙之權利）**
			- 乙 對 甲 有 **367 買受人之義務（甲之權利，乙之義務）**
		- ==**債權讓與**==
			- 把權利給人
			- 上述案例，甲把 367之債權 讓與給 丙
		- ==**債務承擔**==
			- 把義務給人
			- 上述案例，乙把 367之債務 給 丁承擔
		- ==**契約承擔**==
			- 概括把契約的 權利義務 給人
			- 上述案例，乙把 345之契約權利義務 給 戊承擔
			- eg
				- ==**店面之頂讓**==
				- ==**425 買賣不破租賃**==
					- **房東的房屋買賣行為，不影響租客的租屋權益**

- 債權讓與
	- 甲 對 乙 有債權
	- 甲 上述債權 345 買賣契約 丙
	- 若債權分專屬權，則買賣契約有效
	- 甲 294 準物權行為 丙
